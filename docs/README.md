# Documentation

Central documentation hub for PRO Nedvizh AI OS.

## Subdirectories

| Folder | Contents |
|--------|----------|
| [`architecture/`](architecture/) | System design, data models, integration boundaries |
| [`agents/`](agents/) | Agent roles, responsibilities, and handoff rules |
| [`workflows/`](workflows/) | End-to-end process definitions and stage specs |
| [`releases/`](releases/) | Release notes, changelogs, and version history |

## Conventions

- Use Markdown for all docs
- Link to related folders (`/prompts`, `/make`, `/google`) when describing integrations
- Keep workflow docs aligned with the main pipeline defined in the root README
